Создайте приложение по шаблону Windows Forms Application.
Поместите на форму два элемента управления DataGridView.
Первый элемент DataGridView предназначен для вывода данных
таблицы Customers базы данных ShopDB. Также поместите на форму
кнопку, при нажатии на которую во втором элементе DataGridView
должны появляться те строки, которые пользователь изменил в первом
DataGridView.
